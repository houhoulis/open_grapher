Obert Escriptor (Open Grapher)

This is an entirely client-side, in-browser grapher of sequence(s) you add to the URL. No data is sent to any server, this is done entirely on your computer. For example, "open-grapher.html?mysequence=1,2,3" will draw a graph with the points 1, 2, and 3 in order, labeled "mysequence". Try it: open-grapher.html?mysequence=1,2,3.

Any query string you add to the URL in the form "word=number,number,number,,,number,number" will be interpreted as a sequence to be graphed. If two commas are next to each other, that point in the sequence will be skipped. You can add any number of these query strings to the URL -- all series will be graphed. In addition, these special query strings are available:

  • 'títol', or 'title'
  • 'subtítol', or 'subtitle'
  • 'x'
  • 'y'
  • 'inici', or 'start'
  • 'llengua', or 'language'
  • 'aproximada', or 'approximate'

If 'aproximada' (or 'approximate') is set to anything in the query string, then each series of data will be graphed accompanied by best-fit lines calculated via Theil-Sen and Linear Least Squares. I made this page to explore best-fit lines.

This page will support at least Catalan and English.

Here's an example: Click open-grapher.html?títol=My New Graph&subtítol=A simple graph&some data=4,5,5.7,3,5&some different data=1.3,3.3,2.3,1.3&inici=1&x=day&y=weight for a graph with two series.
Or add the "aproximada" term to add the best-fit lines for each series to the graph. open-grapher.html?títol=My Fancier Graph&subtítol=A graph with best fit lines&some data=4,5,5.7,3,5&some different data=1.3,3.3,2.3,1.3&inici=1&x=day&y=weight&aproximada=123.

Edit the URL in the browser's location bar to define your own graph!
